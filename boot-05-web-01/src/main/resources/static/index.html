测试REST风格请求

    car/3/owner/李四
  • @PathVariable（路径变量）
  • @RequestHeader（获取请求头）
    • car/3/owner/lisi?age=18&interest=basket&interest=game
  • @RequestParam（获取请求参数）
  • @CookieValue（获取cookie值）
  • @RequestBody（获取请求体【POST表单】）
    • 测试@RequestBody获取数据
  • @RequestAttribute（获取request域属性）
  • @MatrixVariable（获取矩阵变量：应该绑定在路径中）
    • /cars/{path}?xxx=yyy&aaa=bbb 称之为查询字符串queryString，获取使用@RequestParam
    矩阵变量应该绑定在路径中，看成一个整体 /car/{path;low=34,brand=byd,audi,yd}为矩阵变量 补充：页面开发如果cookie被禁用了，session中的内容如何使用： 可以通过url重写：/abc;jsessionid=xxx把cookie中的值通过矩阵变量传递过来
