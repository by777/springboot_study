  • @PathVariable（路径变量）
  • @RequestHeader（获取请求头）
    • car/3/owner/lisi?age=18&interest=basket&interest=game
  • @RequestParam（获取请求参数）
  • @CookieValue（获取cookie值）
  • @RequestBody（获取请求体【POST表单】）
    • 测试@RequestBody获取数据
  • @RequestAttribute（获取request域属性）
  • @MatrixVariable（获取矩阵变量：应该绑定在路径中）
    • /cars/sell;low=34;brand=byd,audi,yd /cars/{path}?xxx=yyy&aaa=bbb 称之为查询字符串queryString，获取使用@RequestParam(
    此外，还有另外一种获取的方式，矩阵变量 矩阵变量应该绑定在路径中，看成一个整体绑定在路径变量中 /car/{path;low=34;brand=byd,audi,yd}为矩阵变量 之前的面试题： 页面开发如果cookie被禁用了，session中的内容如何使用： cookie，session的机制时，给session中保存了一个东西，session.set("a",b)，每一个人每一个session都有一个sessionID、叫jsessionid。 jsessionid保存至cookie里面。而cookie又是每次发请求都会携带，服务器按照jsessionid找到这个session对象。 然后调用session对象的get方法，找到session里面的内容。如果cookie被禁用了，jsessionid由于存在cookie里面，jsessionid携带不上。 这样cookie上带不了，就用矩阵变量的方式 可以通过url重写：/abc;jsessionid=xxx把cookie中的值通过矩阵变量传递过来。 真实路径是/car/sell;low=34,brand=byd,audi,yd;
    矩阵变量，有两个age的值，如何获取 /boss/1;age=20/2;age=10
